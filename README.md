# Interpolator — Linear Interpolation Calculator

เครื่องมือคำนวณค่าระหว่างสองจุดด้วย Linear Interpolation

## ใช้ทำอะไร
หาค่า Y ที่ไม่รู้ เมื่อรู้ค่า X และจุดอ้างอิงสองจุด  
ใช้ได้ในงาน Engineering, 3D Animation, Physics Simulation

## วิธีใช้
1. กรอก x1, y1 (จุดต่ำสุดที่รู้)
2. กรอก x2 (ค่าที่ต้องการหา y)
3. กรอก x3, y3 (จุดสูงสุดที่รู้)
4. กด Calculate

## สูตรที่ใช้
y2 = y1 + (x2 - x1) × (y3 - y1) / (x3 - x1)
